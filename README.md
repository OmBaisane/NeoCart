NeoCart is a **full-stack PHP & MySQL e-commerce web application** with admin panel, shopping cart, orders, and AJAX features.

## Features
- User authentication (login/register)
- Product listing and details
- Add to cart and checkout
- Admin dashboard to manage products, users, and orders
- AJAX-based search and cart updates
- Responsive design for all devices

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Backend:** PHP, MySQL
- **AJAX:** For dynamic cart and search
- **Server:** XAMPP (Apache + MySQL)

## How to Run Locally
1. Install **XAMPP** and start Apache & MySQL.
2. Copy `NeoCart` folder to `htdocs`.
3. Create a database in phpMyAdmin.
4. Update `config/db.php` with your local DB credentials.
5. Open `http://localhost/NeoCart/` in browser.
