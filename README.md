## Project Overview
NeoCart is a full-stack e-commerce web application developed using PHP and MySQL.  
It includes user authentication, product management, cart functionality, order processing, and an admin dashboard.

## Key Features
- User registration & login
- Product listing & search
- Cart & checkout system
- Admin panel for managing products & orders
- AJAX-based interactions

## Tech Stack
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap
